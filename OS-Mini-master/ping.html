ping command:

ping is a computer network administration software utility used to test the reachability of a host on an Internet Protocol (IP) network. It measures the round-trip time for messages sent from the originating host to a destination computer that are echoed back to the source.
In cases of no response from the target host, most implementations display either nothing or periodically print notifications about timing out. Possible ping results indicating a problem include the following:

  1. H, !N or !P: host, network or protocol unreachable
  2. S: source route failed
  3. F: fragmentation needed
  4. U or !W: destination network/host unknown
  5. I: source host is isolated
  6. A: communication with destination network administratively prohibited
  7. Z: communication with destination host administratively prohibited
  8. Q: for this ToS the destination network is unreachable
  9. T: for this ToS the destination host is unreachable
  10. X: communication administratively prohibited
  11. V: host precedence violation
  12. C: precedence cutoff in effect
