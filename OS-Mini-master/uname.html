uname command:

uname (short for unix name) is a computer program in Unix and Unix-like computer operating systems that prints the name, version and other details about the current machine and the operating system running on it.

  1. -a: (i.e., all)tells uname to provide the following information: the name of the kernel, network node host name (e.g., localhost.localdomain), kernel version number and release level (e.g., 2.4.20-6), kernel release date, machine hardware name, CPU (central processing unit) type, hardware platform and operating system name.
  2. -n: provides network node host name
  3. -r: provides kernel version number and release level
  4. -v: provides date of release of the kernel version
  5. -m: provides machine hardware name
  6. -p: provides CPU type (not available on some systems)
  7. -i: provides general hardware platform
  8. -o: provides operating system name
