nano,vi command:

nano and vi are already installed text editors in the Linux command line.
The nano command is a good text editor that denotes keywords with color and can recognize most languages. And vi is simpler than nano.You can create a new file or modify a file using this editor.
For example, if you need to make a new file named "check.txt", you can create it by using the command “nano check.txt”. You can save your files after editing by using the sequence Ctrl+X, then Y (or N for no).
You can also use jed instead of nano.
