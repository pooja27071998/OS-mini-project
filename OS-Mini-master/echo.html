echo command:

The "echo" command helps us move some data, usually text into a file.
For example, if you want to create a new text file or add to an already made text file, you just need to type in, “echo hello, my name is xyz >> new.txt”. You do not need to separate the spaces by using the backward slash here, because we put in two triangular brackets when we finish what we need to write.
It is a command typically used in shell scripts and batch files to output status text to the screen or a file, or as a source part of a pipeline.
